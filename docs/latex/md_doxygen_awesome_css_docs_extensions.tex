To allow the user to easily navigate from one document to another, \char`\"{}\+Next\char`\"{} and \char`\"{}\+Previous\char`\"{} buttons can be added at the end of a Markdown document.\hypertarget{md_doxygen_awesome_css_docs_extensions_autotoc_md18}{}\doxysubsection{Installation}\label{md_doxygen_awesome_css_docs_extensions_autotoc_md18}

\begin{DoxyItemize}
\item The navigation must be inside a Markdown table with 1-\/2 columns.
\item The alignment of the column defines the alignment of the arrow on the navigation button.
\item the table must be wrapped inside a {\ttfamily $<$div$>$} with the class {\ttfamily section\+\_\+buttons}.
\end{DoxyItemize}
